How to use Cli Mode


First go to that folder in which you downloaded that file, In this case I downloaded file in default download path




then open "Dr.Win" application



allow the application to get administator previlages



When the application starts, all the versions of windows will shown on your screen, then type index of your windows version and hit Enter and your windows activation is start.
In this case i actvate window 10 Professional so i type 5.



Click 'ok'



Click 'ok'



Again Click 'ok' and your window is activated successfully



Type 2 and hit Enter to exit the application


How to use GUI Mode


First go to that folder in which you downloaded that file, In this case I downloaded file in default download path




Extract the Dr.Win zip file





Open the Dr.Win folder



then Run Dr.Win application



allow the application to get administator previlages



Select windows version



In this case i activate window 10 Professional so i choose Professional.



Click on start Button.



Click 'ok'



When you see this Cmd window again and again just ignore it and close it.



Click 'ok'



Again Click 'ok' and your window is activated successfully
